\# Simple Video Frame Extractor



\## 1. Project Overview

\*\*Python과 OpenCV를 활용한 초간단 동영상 프레임 추출기\*\*입니다.

영상 파일(.mp4, .avi)을 입력받아 지정된 간격으로 이미지를 자동 저장해줍니다.

머신러닝 데이터셋 구축이나 영상 캡처가 필요할 때 유용하게 사용할 수 있습니다.



\- \*\*Key Features:\*\*

&nbsp;   - `OpenCV` 기반의 빠르고 가벼운 동작

&nbsp;   - `tqdm`을 이용한 직관적인 진행바 제공

&nbsp;   - 원하는 프레임 간격(Interval) 설정 가능
